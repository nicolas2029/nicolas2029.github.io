Detalles de Punto y Coma

Descripción del proyecto

Nuestra aplicación Web tiene como objetivo automatizar y facilitar el flujo de trabajo para los empleados, usuarios y personal administrativo de una cadena de restaurantes. Se implementaron múltiples tipos de usuarios y diferentes establecimientos en localidades reales, se agregó el sistema de pedidos físicos y a domicilio, se puede obtener información de los pedidos realizados para su análisis por los administradores, los cuales pueden gestionar los productos y establecimientos, además desarrollamos un sistema de clasificación de imágenes para facilitar la búsqueda y registro de productos.

Principales contribuciones al proyecto

Durante aproximadamente 7 meses me encargue principalmente del diseño, desarrollo, documentación y testeo del Backend, las bases de datos y la infraestructura, además de la gestión de actividades a los demás miembros del equipo, en donde asignaba, revisaba y unía las funcionalidades implementadas mediante Git en Github, organizaba reuniones virtuales para debatir y proponer soluciones o estrategias a las necesidades del sistema, brinde asesorías sobre el uso de Git y la comunicación del Backend con el Frontend, por último implemente pipelines en Github Actions para desplegar la aplicación de forma automática en GKE y ofrecer un desarrollo continuo.

A continuación se muestra un listado de mis principales aportes al desarrollo del observatorio:

  • Diseño de diagramas relacionales y diagramas entidad relación para las bases de datos
  • Diseño de diagramas de caso de uso
  • Obtención de requerimientos funcionales y no funcionales
  • Diseño de la aplicación distribuida en microservicios
  • Administración y desarrollo de las bases de datos en PostgreSQL y MongoDB
  • Documentación de las APIs en Postman y creación de un servidor Mock
  • Desarrollo e implementación de los módulos de ventas, usuarios y direcciones
  • Ayude en el desarrollo del módulo de información y clasificación mediante IA
  • Asignación y supervisión de las actividades en Jira
  • Testeo del Backend mediante pruebas unitarias y de integración usando Testify y Mockery
  • Integración del código mediante Git
  • Administración, desarrollo y optimización de costos de la infraestructura en GKE
  • Comunicación entre microservicios mediante GRPC y con servicios externos mediante REST
  • Ayude a realizar la comunicación del Backend con el Frontend
  • Administración y gestión de la seguridad de los repositorios, servicios externos y la aplicación
