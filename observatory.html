Detalles del Observatorio Tecnológico

Descripción del proyecto

El observatorio tecnológico IDITpyme de CUCEA es una plataforma de vigilancia tecnológica que utiliza inteligencia artificial y humana para la generación de oportunidades de negocios, lo que permite a las empresas mejorar su competitividad en el mercado.

Tecnologías utilizadas y frameworks

  • Java - Sprint Boost
  • Angular
  • HTML, CSS, JS
  • TypeScript
  • WordPress
  • PostgreSQL
  • Git y Github
  • Peticiones de tipo Rest

Principales contribuciones al proyecto

Durante mi estancia de 7 meses (07/03/2022 - 29/9/2022) formando parte de un equipo de cinco integrantes para el desarrollo del observatorio tecnológico, realice varios aportes, en mayor medida en el backend utilizando Java con Sprint Boost, pero también utilice Angular y Type Script para realizar las vistas y conectar el frontend y el backend a la vez que realizaba su respectiva documentación, por último realice cambios en la base de datos de PostgreSQL para implementar, corregir y optimizar el funcionamiento de los módulos.

A continuación se muestra un listado de mis principales aportes al desarrollo del observatorio:

  • Desarrolle e integre el módulo de consultorías.
  • Detecte y corregi errores en el inicio de sesión.
  • Brinde asesorías a mis compañeros de equipo para realizar la comunicación entre el frontend y el backend.
  • Agregue y actualice tablas de las bases de datos en PostgreSQL
  • Ayude en el diseño de la landing page en WordPress
  • Ayude a desarrollar los módulos de:
    • Publicaciones y patentes.
    • Proyectos y Líneas de investigación
    • Ofertas tecnológicas
